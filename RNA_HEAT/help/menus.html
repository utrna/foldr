RNA HEAT: Menus



File

Open RNA File: Updates the display with the contents of an RNA data source (such as a '.bpseq' file).  If the option to overlay is not chosen then any previously-opened file is closed. If an overlay is requested then the new RNA appears on top of the previous one, in a different color.

Open Data File: Opens a data file in a separate display, keeping the RNA display untouched.  This can be used to show plain text files, for example.

Run Script: Runs commands from a JavaScript ('.js') file, such as a series of filters. Currently, such scripts must make use of a variable named "rheat". See scripting for more information.

Run Program: Runs external programs, sending output to the current experiment tree.  Use Preferences to specify the top of the tree.  The output of the program appears in a "log.txt" file in the current experiment directory.

Close Window: Hides the frontmost window that has a close box. Note that most RNA HEAT windows will always be visible but this can be used to discard other windows (such as this help page).

Close RNA: Clears the display and all other windows. Note that if you open a new file, any previously-displayed file is closed implicitly.

Save As: Exports a picture of the current display as an image file (such as '.png'). The image file will reflect what is displayed, including any filters or selections. Note that if the display has been zoomed, the image file will be much larger.

Exit (or Quit): Ends the program.

Edit

Undo: Reverses the effects of the most recent action, if possible.  A series of Undo commands is possible, as each previous state is saved as a separate file (see Preferences).

Preferences: Allows customization of behavior, such as the directory to start from when opening files.  These settings are saved as scripting commands in your home folder, under ".rheat/prefs.js".

Filters

Basepair Filter: Excludes helices that do not match selected base-pair values, such as 'C-G'.  If another RNA overlay is being shown, the filter applies to both (unlike other filters).

Helix Length Filter: Excludes helices with a span more or less than the specified range. One of the values may be omitted to include everything in that direction (for instance, omitting the upper value will show helices greater than or equal to the minimum length).

Diagonal Filter: Excludes helices whose distance from the diagonal line (2D view) is outside the given range. One of the values may be omitted to include everything in that direction (for instance, omitting the upper value will show helices that are at least as far from the diagonal as the minimum distance).

AA / AG Ends Filter: Excludes helices that do not have AA or AG “just past” their ends.

E-Loop Filter: Excludes helices unless they have AAG at 3' start, AUG at 5' start, GAA at 3' end, and AUG at 5' end.

Energy Filter: Excludes helices that have an energy value outside the given range. One of the values may be omitted to include everything in that direction (for instance, omitting the upper value will show helices with energy values greater than or equal to the minimum value).

Complex Distance Filter: Excludes helices with simple or complex distances that are greater than the specified maximums.

View

2D: Changes to a two-axis display mode, with a diagonal line.

Flat: Changes to a single-axis display mode.

Zoom Out: Shows more of the display, at a reduced size.

Zoom In: Shows less of the display, at an increased size.

Zoom to Fit: Shows the entire display, at a size that fills the window.

Window

Minimize Window: Reduces the frontmost window to a miniature form. Equivalent to clicking the minimization button on the window frame.

Zoom Window (or Maximize/Restore Window): Toggles the frontmost window between its two main sizes. Equivalent to clicking the maximizing button on the window frame.

Select Next Window: Brings another window to the front.

Select Previous Window: Brings another window to the front (reverse order).

Display Window: Brings the Display Window to the front.  For more information, see the windows page.

Controls: Brings the Controls to the front.  For more information, see the windows page.

Filter History: Brings the Filter History to the front.  For more information, see the windows page.

Helix Info: Brings the Helix Info to the front.  For more information, see the windows page.



Help

Contents: Displays this help window.

Go Back: Displays the previously-visited help page, if a link was followed.

About: Shows a window with information on the authors of RNA HEAT.
