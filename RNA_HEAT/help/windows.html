RNA HEAT: Windows



Display

Shows the effects of any control settings and constraints. The display may be clicked with the mouse to show coordinates and select a helix.

Constraint History

The JavaScript commands that are equivalent to constraint actions that have been performed.  For reuse, these commands can be copied into a script.  They are also automatically saved to a script (in your home folder, under ".rheat/history.js").  If you click “Undo Last” or use the Undo command, the last constraint command in the list is removed and the display rolls back to its previous state.

RNA File Info

Shows a summary of the file that was opened and the available helices.

Helix Info

Information on any selected helix or clicked location.
