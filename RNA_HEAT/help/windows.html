RNA HEAT: Main Window Panels



Display

The display shows the helices of the current RNA, after applying any constraints or control settings.  The display may be clicked with the mouse to show coordinates and select a helix.  The zoom controls may be used as short-cuts for commands in the View menu. The jump fields can be used to scroll the display to a particular position, using a base-pair number (for example, if the RNA sequence has 2900 base-pairs, entering "1450" and clicking Go will jump to the center of the display).  If X or Y are omitted when clicking Go, the values are assumed to be the same.

View Frame

This miniature window appears only when some part of the display is not visible due to zooming and/or scrolling.  Its white rectangle shows which part of the entire RNA is currently visible.  The frame may be moved to another location by dragging its title bar.

RNA File Info

The top-left region shows a summary of the RNA file that is open. The Current count depends on any applied Constraints, the Total count reflects only the last Base-Pair Filter and the Actual count refers to the helices from the original input file (the “correct” helices shown in the top-right triangle of the display).

Selected Helix

When you click a helix line, this area shows more information about the helix (such as its start and end points, base-pairs, and energy).  Two helices that appear mirrored on the diagonal of the 2D display often have similar or identical information when selected (for example, the helix length may vary between the two).

Constraint History

The JavaScript commands that are equivalent to constraint actions that have been performed.  For reuse, these commands can be copied into a script.  They are also automatically saved to a script (in your home folder, under ".rheat/constraints.js").  If you click “Undo Last” or use the Undo command, the last constraint command in the list is removed and the display rolls back to its previous state.
