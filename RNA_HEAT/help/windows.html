RNA HEAT: Windows



Controls

Shows a summary of the file that was opened and the available helices.  Also provides short-cuts for some menu commands.

Display

Shows the effects of any control settings and applied filters.  The display may be clicked with the mouse to show coordinates and select a helix.

Filter History

A series of JavaScript commands that are equivalent to filter actions that have been performed.  For reuse, these commands can be copied into a script.  They are also automatically saved to a script (in your home folder, under ".rheat/history.js").  If you click “Undo Last” or use the Undo menu command, the last filter command in the list is removed and the display rolls back to its previous state.

Helix Info

Information on any selected helix or clicked location.
